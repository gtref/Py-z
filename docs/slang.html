About Slang Mode

This mode provides an alternative set of keywords and CLI commands for a more fun and "online" coding experience.

Activation

Activate slang mode by passing the --slang flag as the first argument to the super CLI.

super --slang my_slang_file.super
super --slang cook my_lib.super
